# AgendaEz

## -- FOR THE PLYCKY HACKATHON GRP 2 --

**AgendaEz** is your all-in-one organizer and virtual pet companion!  
Manage your schedule, earn points by completing tasks, and use those points to care for and customize your digital pet.  
Switch between beautiful dark and light modes, and enjoy a playful, productive experience.

---

## ✨ Features

- **Task Organizer:**  
  Add, view, and complete tasks with a clean, calendar-inspired interface.

- **Points System:**  
  Earn points for completing tasks. Spend points to feed, play with, or buy new pets.

- **Virtual Pet:**  
  Choose, name, and care for your pet. Each pet has unique stats and can be renamed individually.

- **Plan Selector:**  
  Switch between Free, Pro, and Premium plans to unlock more task slots.

- **Dark/Light Mode:**  
  Toggle between a soothing dark mode and a gentle, readable light mode.  
  Your preference is saved and synced across all pages.

## 🚀 Getting Started

### 1. **Clone or Download**

```sh
git clone https://github.com/yourusername/agendaez.git
cd agendaez
```

Or [download as ZIP](https://github.com/yourusername/agendaez/archive/refs/heads/main.zip) and extract.

---

### 2. **Run Locally**

> **For best results, use a local server (not `file://`):**

- **VS Code:**  
  Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) and right-click `index.html` → "Open with Live Server".

- **Python:**  
  ```sh
  python3 -m http.server
  ```
  Then open [http://localhost:8000](http://localhost:8000) in your browser.

---

### 3. **Open in Browser**

- Go to `index.html` for the organizer.
- Go to `pet.html` for your pet.
- Go to `homescreen.html` for the dashboard and theme switch.

## 🛠️ Customization

- **Theme:**  
  Use the 🌙/☀️ button on any page to toggle dark/light mode.

- **Plans:**  
  Use the plan selector to switch between Free (5 tasks), Pro (15 tasks), and Premium (unlimited tasks).

- **Pets:**  
  Buy, select, and rename pets. Each pet's name is unique!
